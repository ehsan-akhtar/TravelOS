## Overview

Orbit OS transforms AI interactions from a single chatbot into an intelligent ecosystem of autonomous agents.

Instead of asking one AI to complete a task, specialized agents collaborate in real time:

- Flight Agent
- Research Agent
- Planning Agent
- Verification Agent
- Budget Agent
- Calendar Agent
- Knowledge Agent
- QA Agent

Each agent works independently while sharing memory, reasoning, and execution state across the workflow.
